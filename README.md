# Ural_owl_habitat_model
This repository contains the Rscript for the analysis, the kuenm results and the model lambdas reported in the paper "Habitat Potential and Management Perspectives of the Reintroduced Ural Owl (Strix uralensis) in a Highly Human-Dominated Landscape", published in the Journal "Conservation Science and Practice" (see link to publication in the zenodo record).
